# Sesd Shop

A responsive React frontend with a Node/Express backend.

## Run Locally

Install dependencies:

```bash
npm install
```

Start both applications:

```bash
npm run dev
```

- Frontend: `http://localhost:5173`
- Backend API: `http://localhost:4000/api`

## Build

```bash
npm run build
```

## Server Endpoints

- `GET /api/products`
- `POST /api/contact`
